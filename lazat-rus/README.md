# Getting Started

1. `npm i`
2. `npm run dev`

# Contributing

1. `git clone <url>`
2. `git switch -c some-feature-branch`
3. Edit files
4. `git add .`
5. `git commit -m "some commit"`
6. `git pull origin main --rebase`
7. Resolve conflicts (if any)
8. `git push -u origin some-feature-branch`
9. Go to GitLab repo
10. Create a PR
